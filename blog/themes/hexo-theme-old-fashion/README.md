# hexo-theme-old-fashion

old fashion theme for hexo

```shell
npm install hexo-generator-sitemap --save

# Add To _config.yml
sitemap:
  path:
  - sitemap.xml
```

```shell
npm install hexo-generator-feed --save

# Add To _config.yml
feed:
  enable: true
  type:
  - atom
  - rss2
  path:
  - atom.xml
  - rss2.xml
```
